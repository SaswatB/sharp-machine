About us...

The Sharp Machine is a project started by Samad Farooqui, Saswat Bhattacharya, Toshi Piazza, and Ashwin Krishna. We wanted to make something that professionals could use in order to understand their data in a more convenient way. Click here to try it out!

The Modular System

This website allows anyone to add their own algorithm in order to parse their own data, or use some that are already pre-made into the website. If you would like to use your own algorithm, just upload your file with the proper structure and train the website. The built in algorithms are linear perception with pocket, neural networking, and linear regression.
